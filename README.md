# TraceNet Blockchain

Complete blockchain ecosystem with DPoA consensus, social features, and LT token economy.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📦 Features

- ✅ Delegated Proof of Activity (DPoA) Consensus
- ✅ Multi-wallet System
- ✅ Social Media Integration
- ✅ LT Token Economics
- ✅ Supabase Off-chain Storage
- ✅ Automatic Blockchain Backups
- ✅ CI/CD with GitHub → Google Cloud

## 🌐 Deployment

Automatic deployment on every push to main branch!
