📚 API Dokümantasyonu

👤 Kullanıcı API'leri

POST /api/user/create
Yeni kullanıcı oluştur (otomatik cüzdan ve airdrop ile)
Parametre Tip Zorunlu Açıklama
nickname string ✅ Evet Benzersiz kullanıcı adı (3-20 karakter)
name string ✅ Evet Ad
surname string ❌ Hayır Soyad
birthdate string ❌ Hayır Doğum tarihi (YYYY-MM-DD)
{ "user": { "user_id": "...", "nickname": "alice", "name": "Alice" }, "wallet": { "wallet_id": "TRN...", "public_key": "..." }, "mnemonic": "word1 word2 ... word24", "airdrop_amount": "0.00625 LT" }
GET /api/user/nickname/:nickname
Nickname ile kullanıcı bilgilerini getir
GET /api/user/search?query=...
Kullanıcı ara (nickname veya isimde arama)

📝 İçerik API'leri

POST /api/content/create
Yeni içerik paylaş (ÜCRETSİZ)
Parametre Tip Zorunlu Açıklama
wallet_id string ✅ Evet İçerik sahibinin cüzdan ID'si
content_type string ✅ Evet TEXT, VIDEO, PHOTO, TWEET
description string ❌ Hayır İçerik açıklaması
content_url string ❌ Hayır Medya URL'si (VIDEO/PHOTO için)
GET /api/content/feed?limit=20
İçerik feed'i getir (en yeni içerikler)
GET /api/content/:contentId
Belirli bir içeriği getir (beğeni/yorum sayıları ile)
GET /api/content/user/:walletId
Kullanıcının tüm içeriklerini getir

❤️ Sosyal Etkileşim API'leri

POST /api/social/like
İçeriği beğen
💰 Ücret: 0.00001 LT (50% içerik sahibine, 50% blockchain hazinesine)
Parametre Tip Zorunlu Açıklama
wallet_id string ✅ Evet Beğenen kullanıcının cüzdan ID'si
content_id string ✅ Evet Beğenilen içeriğin ID'si
{ "success": true, "tx_id": "...", "fee_breakdown": { "total_fee": 1000, "creator_received": 500, "treasury_received": 500 } }
POST /api/social/comment
İçeriğe yorum yap
💰 Ücret: 0.00001 LT (50% içerik sahibine, 50% blockchain hazinesine)
Parametre Tip Zorunlu Açıklama
wallet_id string ✅ Evet Yorum yapan kullanıcının cüzdan ID'si
content_id string ✅ Evet Yorum yapılan içeriğin ID'si
comment_text string ✅ Evet Yorum metni (max 500 karakter)
POST /api/social/follow
Kullanıcıyı takip et (ÜCRETSİZ)
POST /api/social/unfollow
Kullanıcıyı takipten çık (ÜCRETSİZ)
GET /api/social/likes/:contentId
İçeriğin tüm beğenilerini listele
GET /api/social/comments/:contentId
İçeriğin tüm yorumlarını listele

💼 Cüzdan API'leri

POST /api/wallet/create
Yeni cüzdan oluştur
⚠️ GÜVENLİK UYARISI: Bu endpoint yalnızca güvenli, client-side uygulamalarda kullanılmalıdır. Mnemonic ve private key'i ASLA paylaşmayın veya sunucu tarafında saklamayın!
GET /api/wallet/:walletId
Cüzdan bakiyesi ve bilgilerini getir
GET /api/wallet/list/:userId
Kullanıcının tüm cüzdanlarını listele
