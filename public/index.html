👤 Kullanıcı API'leri

POST /api/user/create
Yeni kullanıcı profili oluştur (nickname, name, surname)
{ "nickname": "alice", "name": "Alice", "surname": "Johnson" }
GET /api/user/nickname/:nickname
Nickname ile kullanıcı bilgisi getir
GET /api/user/search?q=query
Kullanıcı ara

📝 İçerik API'leri

POST /api/content/create
İçerik paylaş (VIDEO, PHOTO, TEXT, TWEET)
{ "wallet_id": "LT_...", "content_type": "TEXT", "description": "Merhaba dünya!", "tags": ["ilk", "post"] }
GET /api/content/feed?limit=20
İçerik akışını getir
GET /api/content/:contentId
İçerik detayını getir (beğeni, yorum sayısı ile)
