📡 API Endpoints

🌐 Base URL

https://tracenet-blockchain-bbzxtm72vq-uc.a.run.app

🔗 Blockchain RPC

GET /health - Server health check
GET /rpc/status - Blockchain status
GET /rpc/block/:index - Block details
GET /rpc/transaction/:txId - TX info
GET /rpc/balance/:walletId - Wallet balance
POST /rpc/sendRawTx - Submit transaction

💰 Economy API

GET /economy/tokenPrice - Token price & market data
GET /economy/treasury - Treasury stats
GET /economy/distribution - Token distribution

💼 Wallet API

POST /api/wallet/create - Create new wallet
GET /api/wallet/list/:userId - List user wallets
GET /api/wallet/:walletId - Wallet details

📘 JavaScript Example

const API = 'https://tracenet-blockchain-bbzxtm72vq-uc.a.run.app';

// Get blockchain status
fetch(`${API}/rpc/status`)
  .then(r => r.json())
  .then(data => {
    console.log('Blocks:', data.blockchain.blockCount);
    console.log('Transactions:', data.blockchain.totalTransactions);
  });

// Get specific block
fetch(`${API}/rpc/block/0`)
  .then(r => r.json())
  .then(block => console.log('Block:', block));

// Create wallet
fetch(`${API}/api/wallet/create`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ user_id: 'user123' })
})
  .then(r => r.json())
  .then(wallet => console.log('New Wallet:', wallet));

🐍 Python Example

import requests

API = 'https://tracenet-blockchain-bbzxtm72vq-uc.a.run.app'

# Get blockchain status
status = requests.get(f'{API}/rpc/status').json()
print(f"Blocks: {status['blockchain']['blockCount']}")

# Get block
block = requests.get(f'{API}/rpc/block/0').json()
print(f"Genesis Hash: {block['hash']}")

# Create wallet
wallet = requests.post(
    f'{API}/api/wallet/create',
    json={'user_id': 'user123'}
).json()
print(f"Wallet: {wallet['wallet']['wallet_id']}")
