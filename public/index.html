Özellikler

⚡

DPoA Consensus

Delegated Proof of Activity ile hızlı ve güvenli konsensus. 7 validator imzası ile transaction onayı.

💰

LT Token Economy

100M sabit supply, dinamik fiyatlama, otomatik airdrop ve sosyal interaction ödülleri.

👥

Social Features

On-chain sosyal medya: posts, likes, comments, follows. Her interaction blockchain'de kayıtlı.

Multi-Wallet

ED25519 + BIP39 mnemonic wallet sistemi. Her kullanıcı birden fazla wallet oluşturabilir.

💾

Supabase Integration

Off-chain data storage, otomatik blockchain backup, real-time subscriptions.

🚀

Auto Deploy

GitHub push → Google Cloud otomatik deployment. CI/CD pipeline ile sürekli entegrasyon.

API Endpoints

Blockchain RPC

GET /rpc/status - Node durumu
POST /rpc/sendRawTx - Transaction gönder
GET /rpc/block/:id - Block bilgisi
GET /rpc/transaction/:txId - Transaction bilgisi
GET /rpc/balance/:walletId - Wallet bakiyesi

Token Economy

GET /economy/tokenPrice - Token fiyatı ve market data
GET /economy/userValue/:userId - Kullanıcı token değeri (USD)
GET /economy/treasury - Treasury istatistikleri
GET /economy/distribution - Token dağılımı

Örnek Kullanım

// Node status
fetch('https://your-node.run.app/rpc/status')
  .then(r => r.json())
  .then(data => console.log(data));

// Token price
fetch('https://your-node.run.app/economy/tokenPrice')
  .then(r => r.json())
  .then(data => console.log(data.price_usd));

Node Kurulumu

Gereksinimler

  • Node.js 18+
  • npm veya yarn
  • 4GB RAM minimum
  • 10GB disk alanı

Kurulum Adımları

# Repository'yi clone et
git clone https://github.com/LodosLawson/TraceNet.git
cd TraceNet

# Dependencies kur
npm install

# .env dosyasını yapılandır
cp .env.example .env
# .env dosyasını düzenle

# Build
npm run build

# Node'u başlat
npm run dev

Environment Variables

NODE_ENV=production
PORT=3000
GENESIS_VALIDATOR_ID=your-validator-id
BLOCK_TIME_MS=5000
SUPABASE_URL=your-supabase-url
SUPABASE_SERVICE_ROLE_KEY=your-key

{

  "stake_amount": 1000000000
}

// Heartbeat gönder (her 30 saniyede)

{
  "validator_id": "your-unique-id",
  "timestamp": 1234567890
