⛓️ Blockchain

GET /chain
Tüm blockchain zinciri
GET /rpc/block/:indexOrHash
Belirli bir bloğu getir (index veya hash)
GET /rpc/transaction/:txId

POST /rpc/sendRawTx

💼 Cüzdan

📖 Kod Örnekleri

🟨 JavaScript Örneği

const API = 'https://tracenet-blockchain-bbzxtm72vq-uc.a.run.app';

// Blockchain durumunu al
fetch(`${API}/rpc/status`)
  .then(r => r.json())
  .then(data => {
    console.log('Blok sayısı:', data.blockchain.blockCount);
    console.log('İşlem sayısı:', data.blockchain.totalTransactions);
  });

// Belirli bir bloğu getir
fetch(`${API}/rpc/block/0`)
  .then(r => r.json())
  .then(block => console.log('Genesis Block:', block));

// Yeni cüzdan oluştur
fetch(`${API}/api/wallet/create`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ user_id: 'user123' })
})
  .then(r => r.json())
  .then(wallet => console.log('Yeni Cüzdan:', wallet));

🐍 Python Örneği

import requests

API = 'https://tracenet-blockchain-bbzxtm72vq-uc.a.run.app'

# Blockchain durumu
status = requests.get(f'{API}/rpc/status').json()
print(f"Blok sayısı: {status['blockchain']['blockCount']}")

# Blok getir
block = requests.get(f'{API}/rpc/block/0').json()
print(f"Genesis Hash: {block['hash']}")

# Cüzdan oluştur
wallet = requests.post(
    f'{API}/api/wallet/create',
    json={'user_id': 'user123'}
).json()
print(f"Cüzdan ID: {wallet['wallet']['wallet_id']}")
