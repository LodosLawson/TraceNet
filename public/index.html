📖 TraceNet Blockchain Hakkında

TraceNet Nedir?
TraceNet, LockTrace Network tarafından geliştirilen sosyal medya odaklı bir blockchain altyapısıdır. Kullanıcılar içerik paylaşabilir, beğeni ve yorum yaparak etkileşimde bulunabilir ve bu etkileşimlerden token kazanabilir.
🎯 Temel Özellikler:
• Kullanıcı profilleri (nickname tabanlı)
• İçerik paylaşımı (Video, Foto, Text, Tweet)
• Sosyal etkileşimler (Beğeni, Yorum, Takip)
• %50 ücret paylaşımı (içerik sahibine kazanç)
• 100M LT sabit arz (deflasyonist model)

👤 Kullanıcı API'leri

POST /api/user/create
Yeni kullanıcı profili oluştur ve otomatik airdrop al

İstek Parametreleri:

Parametre Tip Zorunlu Açıklama
nickname string ✅ Evet 3-20 karakter, alfanümerik + alt çizgi
name string ❌ Hayır Kullanıcının adı
surname string ❌ Hayır Kullanıcının soyadı
birth_date number ❌ Hayır Unix timestamp (13+ yaş kontrolü)

Örnek İstek:

{ "nickname": "alice", "name": "Alice", "surname": "Johnson" }

Örnek Yanıt:

{ "success": true, "user": { "user_id": "uuid", "nickname": "alice", "wallet_id": "LT_alice_001" }, "wallet": {...}, "airdrop_amount": "0.00625 LT" }
GET /api/user/nickname/:nickname
Nickname ile kullanıcı bilgisi ve istatistikleri getir

Örnek Kullanım:

GET /api/user/nickname/alice
GET /api/user/search?q=query&limit=20
Nickname ile kullanıcı ara
GET /api/user/check-nickname/:nickname
Nickname kullanılabilirliğini kontrol et
POST /api/content/create
Yeni içerik paylaş (VIDEO, PHOTO, TEXT, TWEET)

İstek Parametreleri:

Parametre Tip Zorunlu Açıklama
wallet_id string ✅ Evet İçerik sahibinin cüzdan ID'si
content_type enum ✅ Evet VIDEO | PHOTO | TEXT | TWEET
title string ❌ Hayır İçerik başlığı (max 200 karakter)
description string ❌ Hayır İçerik açıklaması (max 5000 karakter)
content_url string ❌ Hayır Medya URL'i (IPFS veya harici)
tags array ❌ Hayır İçerik etiketleri

Örnek İstek:

{ "wallet_id": "LT_alice_001", "content_type": "TEXT", "description": "İlk blockchain postum!", "tags": ["ilk", "post", "blockchain"] }
GET /api/content/:contentId
İçerik detayını getir (beğeni, yorum, paylaşım sayısı ile)

Yanıt İçeriği:

{ "content": {...}, "likes_count": 42, "comments_count": 15, "shares_count": 8 }
GET /api/content/user/:walletId?limit=50
Kullanıcının paylaştığı içerikleri listele
GET /api/content/feed?limit=20&offset=0
Ana içerik akışını getir (en yeni içerikler önce)

❤️ Sosyal Etkileşim API'leri

💰 Ücret Yapısı ve Kazanç Modeli:
• Beğeni: 0.00001 LT → %50 içerik sahibine (0.000005 LT kazanç!), %50 blockchain'e
• Yorum: 0.00001 LT → %50 içerik sahibine (0.000005 LT kazanç!), %50 blockchain'e
• Mesaj: 0.0000005 LT → %50 alıcıya, %50 blockchain'e
• Takip: ÜCRETSİZ ✅
POST /api/social/like
İçeriği beğen (0.00001 LT ücretli - içerik sahibi kazanır)

İstek Parametreleri:

Parametre Tip Zorunlu Açıklama
wallet_id string ✅ Evet Beğenen kullanıcının cüzdan ID'si
content_id string ✅ Evet Beğenilecek içeriğin hash ID'si

Örnek İstek:

{ "wallet_id": "LT_bob_001", "content_id": "content_hash_123" }

Örnek Yanıt:

{ "success": true, "tx_id": "transaction_hash", "fee_breakdown": { "total_fee": 1000, "creator_received": 500, "treasury_received": 500, "fee_split": "50% creator / 50% treasury" } }
POST /api/social/comment
İçeriğe yorum yap (0.00001 LT ücretli - içerik sahibi kazanır)

İstek Parametreleri:

Parametre Tip Zorunlu Açıklama
wallet_id string ✅ Evet Yorum yapan kullanıcının cüzdan ID'si
content_id string ✅ Evet Yorum yapılacak içeriğin hash ID'si
comment_text string ✅ Evet Yorum metni (max 1000 karakter)
parent_comment_id string ❌ Hayır Yanıt için üst yorum ID'si

Örnek İstek:

{ "wallet_id": "LT_bob_001", "content_id": "content_hash_123", "comment_text": "Harika içerik, çok beğendim!" }
POST /api/social/follow
Kullanıcıyı takip et (ÜCRETSİZ)

İstek Parametreleri:

Parametre Tip Zorunlu Açıklama
wallet_id string ✅ Evet Takip eden kullanıcının cüzdan ID'si
target_wallet_id string ✅ Evet Takip edilecek kullanıcının cüzdan ID'si
POST /api/social/unfollow
Kullanıcıyı takipten çık (ÜCRETSİZ)
GET /api/social/likes/:contentId
İçeriğin tüm beğenilerini listele
GET /api/social/comments/:contentId
İçeriğin tüm yorumlarını listele
GET /api/social/followers/:walletId
Kullanıcının takipçilerini listele
GET /api/social/following/:walletId
Kullanıcının takip ettiklerini listele

JavaScript Örnekleri

// İçerik paylaş const response = await fetch('http://localhost:3000/api/content/create', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ wallet_id: 'LT_alice_001', content_type: 'TEXT', description: 'İlk blockchain postum!' }) }); const data = await response.json(); console.log(data); // İçeriği beğen const likeResponse = await fetch('http://localhost:3000/api/social/like', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ wallet_id: 'LT_bob_001', content_id: 'content_hash_123' }) }); const likeData = await likeResponse.json(); console.log('Fee breakdown:', likeData.fee_breakdown);

💎 Token Ekonomisi

LT Token Detayları:
• Toplam Arz: 100,000,000 LT (sabit, artırılamaz - deflasyonist model)
• Başlangıç Fiyat: $0.00001
• Başlangıç Market Cap: $1,000
• Airdrop: 0.00625 LT (yeni kullanıcılara otomatik)
• Mesaj Ücreti: 0.0000005 LT
• Beğeni/Yorum: 0.00001 LT (50% içerik sahibine, 50% blockchain'e)
💰 Kazanç Modeli:
1. İçerik oluştur (ücretsiz)
2. Beğeni al → 0.000005 LT kazan (her beğeni için)
3. Yorum al → 0.000005 LT kazan (her yorum için)
4. Pasif gelir: Ne kadar çok etkileşim, o kadar çok kazanç!
